extract-sbix
============

Extracts PNGs from *Apple Color Emoji.ttc* (or other SBIX fonts).

Usage
-----



On your Mac:

1. Checkout the code.
2. a. If you have [uv](https://docs.astral.sh/uv/), `uv run extract-sbix.py`
   b. Otherwise, create a Python venv, install deps with `pip install -e .`, run `python extract-sbix.py`.


Look in `images/`.
